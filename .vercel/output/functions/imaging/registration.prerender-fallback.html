Key Formulas

Rigid: x' = R·x + t (rotation + translation)

Affine: x' = A·x + t (linear + translation)

Elastic: x' = x + u(x) (displacement field)

NCC: NCC(I₁,I₂) = Σ(I₁−μ₁)(I₂−μ₂) / √[Σ(I₁−μ₁)² · Σ(I₂−μ₂)²]

MI: MI(I₁,I₂) = H(I₁) + H(I₂) − H(I₁,I₂)

RMSE: RMSE = √(Σ(x̂ᵢ − xᵢ)² / N)
